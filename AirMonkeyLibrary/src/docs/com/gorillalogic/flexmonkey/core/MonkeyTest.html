Classpublic class MonkeyTest
InheritanceMonkeyTest MonkeyCommand flash.events.EventDispatcher

A MonkeyTest is an element in a hierarchical collection of tests. Its children are executable MonkeyCommands, including MonkeyTests, UIEventMonkeyCommands, PauseMonkeyCommands and VerifyMonkeyCommands.
